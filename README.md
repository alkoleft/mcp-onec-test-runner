# MCP YAXUnit Runner

Модуль для запуска тестов YAXUnit в контексте MCP (Model Checking Platform).

## Требования

- JDK 17 или выше
- Gradle 8.5 или выше

## Сборка

```bash
./gradlew build
```

## Запуск тестов

```bash
./gradlew test
```

## Проверка стиля кода

```bash
./gradlew ktlintCheck
```

## Анализ покрытия кода

```bash
./gradlew jacocoTestReport
```
